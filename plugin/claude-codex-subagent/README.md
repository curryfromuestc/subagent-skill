# claude-codex-subagent Plugin

Claude Code plugin package that embeds a mixed-subagent skill.

## Included components

- `.claude-plugin/plugin.json`
- `skills/spawn-codex-worker/SKILL.md`
- `skills/spawn-codex-worker/scripts/spawn-codex-worker.sh`
- `skills/spawn-codex-worker/scripts/spawn-claude-worker.sh`
- `skills/spawn-codex-worker/references/spawn-workflow.md`

## Purpose

Enable Claude Code main sessions to orchestrate both Codex and Claude sub-agents through wrapper scripts.
